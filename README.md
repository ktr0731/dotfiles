## Installation
### Make all configuration files.
``` sh
$ make all
```

- If `$DOTPATH` does not exist, clone this repository.
- Make symbolic links to `$HOME`.
  
### Help
``` sh
$ make help
```

- Show description of some make commands.
  
  
## Update
``` sh
$ make update
```
